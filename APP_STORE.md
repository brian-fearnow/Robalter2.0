## Description

Robalter is a scoring app for golfers who play competitive formats with friends. It handles the complexity of multi-format rounds so you can focus on playing.

**Game Modes:**
- **Sixes** – Rotating partner game across three 6-hole segments
- **Baseball / 9-Point** – 3-player point game with optional birdie sweeps and double back nine
- **The Wheel** – Designed for 5-player groups, rotating pairs across 6-hole segments
- **Four Ball** – Traditional 2v2 match play with Nassau betting and auto-press support
- **Wolf** – Classic format for 3 or 4 players where the Wolf picks a partner (or goes alone) on each hole
- **Skins** – Real-time multiplayer skins game synced across multiple devices and foursomes
- **Independent Matches** – Flexible head-to-head side bets with custom stakes

**Key Features:**
- Automatic course handicap calculations using USGA-style formulas
- GHIN integration to look up player handicaps and course data
- Real-time multiplayer syncing via live leaderboard
- Built-in course database with support for custom courses
- Mobile-first design optimized for use on the golf course
